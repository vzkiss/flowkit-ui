## What is Flowkit UI?

Patterns teams rebuild on every project - built once, reusable everywhere. Each one targets a concrete problem you often hit when building apps.

## Installation

Add `flowkit-ui` as a registry in your [`components.json`](https://ui.shadcn.com/docs/components-json) to use short names instead of full URLs. The CLI swaps `{name}` for the component id when it downloads the registry file.

```json
{
  "registries": {
    "@flowkit-ui": "https://flowkit-ui.vzkiss.com/r/{name}.json"
  }
}
```
