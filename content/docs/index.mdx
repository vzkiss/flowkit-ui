## What is Flowkit UI?

Each component is a React building block built with shadcn/ui, **[Base UI](https://base-ui.com/)**, **Tailwind CSS**, and **Lucide** icons.

The components here are meant to be **accessible** and **simple to customize**, so you can drop them into production apps without fighting your styling layer. Each one targets a concrete problem you often hit when building apps.

The catalog is still small (today: [Creatable Combobox](/docs/components/creatable-combobox)); it will grow over time.

## Installation

Install components with the [shadcn CLI](https://ui.shadcn.com/docs/cli). To use the short name (`@flowkit/creatable-combobox`, and so on), add Flowkit to your app’s [`components.json`](https://ui.shadcn.com/docs/components-json) under `registries`. The CLI uses that URL as a template: it swaps `{name}` for the component id when it downloads the registry file.

```json
{
  "registries": {
    "@flowkit": "https://flowkit-ui.vzkiss.com/r/{name}.json"
  }
}
```
